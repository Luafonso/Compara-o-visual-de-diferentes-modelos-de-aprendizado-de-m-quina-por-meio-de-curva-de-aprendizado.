# Comparação visual de diferentes modelos de aprendizado de máquinas por meio de curva de aprendizado

Um projeto de comparação através de curva de aprendizado de diferentes modelos de Machine Learning.
Nesse projeto o usuário poderá colocar os próprios dados, treinar os modelos e comparar através da curva de aprendizado.

## Tecnologias 

* Linguagem: Python

* Interface: CustomTkinter ou Streamlit (Ainda decidindo)

* Processamento: Pandas e Scikit-Learn
